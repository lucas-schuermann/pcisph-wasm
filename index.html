This demo was tested on Chrome 107. It should work with any browser that supports WebAssembly threads (SharedArrayBuffer), WebGL2, and OffscreenCanvas. At the time of writing, these features are supported for ~77% of web users. A desktop browser is recommended. 

Browser does not support required features
